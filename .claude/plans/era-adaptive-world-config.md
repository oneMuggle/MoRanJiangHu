# 时代自适应世界观预设方案

## 需求重述

用户选择时代背景后，Step 0 中的世界版图、宗门密度、武力等级、能力类型、王朝/天骄占位符、世界观预设卡片等都应跟随时代自动调整，而不是始终显示武侠风格的选项。

## 核心问题分析

当前 Step 0 的所有字段都是静态的：
- **世界版图**：弹丸之地/九州宏大/无尽位面（武侠味太重）
- **宗门密度**：稀少/适中/林立（非武侠时代不应叫"宗门"）
- **武力等级**：低武/中武/高武/修仙（只有武侠/修仙有意义）
- **能力类型**：传统武侠/修仙/超能力/混合（应该按时代推荐）
- **王朝设定/天骄设定**：占位符文本是武侠风格
- **世界观预设卡片**：传统武侠/修仙世界/高武/低武（纯武侠向）

## 实现方案

### Phase 1: 扩展 `时代配置` 接口

**文件**: `models/system.ts`

在 `时代配置` 接口中新增字段：

```typescript
export interface 时代配置 {
    // ... 现有字段
    默认世界版图?: '弹丸之地' | '九州宏大' | '无尽位面';
    默认组织密度?: '稀少' | '适中' | '林立';
    默认能力类型?: 能力类型;
    默认武力等级?: 武力等级;
    默认王朝占位符?: string;
    默认天骄占位符?: string;
    世界观预设卡片?: Array<{ name: string; overrides: Partial<WorldGenConfig> }>;
}
```

为 5 个内置时代填充默认值：

| 字段 | 古代武侠 | 近代民国 | 现代都市 | 近未来赛博 | 未来科幻 |
|---|---|---|---|---|---|
| 默认世界版图 | 九州宏大 | 九州宏大 | 弹丸之地 | 弹丸之地 | 无尽位面 |
| 默认组织密度 | 林立 | 适中 | 适中 | 林立 | 适中 |
| 默认能力类型 | 传统武侠 | 传统武侠 | 传统武侠 | 超能力线 | 超能力线 |
| 默认武力等级 | 中武 | 低武 | 低武 | 中武 | 高武 |
| 默认王朝占位符 | 群雄逐鹿，王朝末年 | 军阀割据，租界林立 | 资本垄断，寡头竞争 | 企业掌控城市命脉 | 星际联邦，殖民地自治 |
| 默认天骄占位符 | 大争之世，天骄并起 | 新旧交替，人才辈出 | 行业精英，创新先锋 | 赛博传奇，黑客天才 | 基因优化，星际探索者 |

### Phase 2: UI 层时代联动

**文件**: `NewGameWizardContent.tsx`

#### 2.1 时代变更处理函数

```typescript
const 处理时代变更 = (newEraId: string) => {
    const era = 内置时代配置.find(c => c.id === newEraId);
    if (!era) return;
    setWorldConfig(prev => ({
        ...prev,
        时代配置ID: newEraId,
        能力类型: era.默认能力类型 ?? prev.能力类型,
        武力等级: era.默认武力等级 ?? prev.武力等级,
        worldSize: era.默认世界版图 ?? prev.worldSize,
        sectDensity: era.默认组织密度 ?? prev.sectDensity,
        dynastySetting: era.默认王朝占位符 ?? prev.dynastySetting,
        tianjiaoSetting: era.默认天骄占位符 ?? prev.tianjiaoSetting,
    }));
};
```

#### 2.2 标签文案动态化

"宗门密度" 标签根据时代改为：
- 古代武侠 → "宗门密度"
- 近代民国 → "组织密度"
- 现代都市 → "集团密度"
- 近未来赛博 → "企业密度"
- 未来科幻 → "势力密度"

#### 2.3 世界观预设卡片动态化

每个时代展示 4 张不同的预设卡片：
- 古代武侠：传统武侠/修仙世界/高武世界/低武江湖（保持现状）
- 近代民国：乱世枭雄/租界风云/帮派争霸/学堂新潮
- 现代都市：都市精英/底层逆袭/职场风云/平凡生活
- 近未来赛博：企业战争/街头黑客/义体佣兵/数据深渊
- 未来科幻：星际殖民/AI觉醒/基因战争/深空探险

### Phase 3: 能力类型选项时代化

能力类型选项根据时代过滤/重排：
- 古代武侠: 传统武侠(默认) / 修仙体系 / 混合世界
- 近代民国: 传统武侠(默认) / 超能力线 / 混合世界
- 现代都市: 传统武侠 / 超能力线(默认) / 混合世界
- 近未来赛博: 超能力线(默认) / 混合世界
- 未来科幻: 超能力线(默认) / 混合世界 / 修仙体系

### 风险与注意事项

1. **向后兼容**：`时代配置ID` 为空时默认古代武侠，所有新字段均为 optional
2. **用户手动修改优先**：时代变更只设置默认值，使用 `??` 确保只填充空值
3. **能力类型联动**：复用现有的 `处理能力类型变更` 逻辑保持一致性
4. **预设卡片 overrides**：每个预设卡片的 `overrides` 必须是合法的 `WorldGenConfig` 子集

### 文件变更清单

| 文件 | 变更类型 | 说明 |
|---|---|---|

### 复杂度评估

**低复杂度** — 主要是数据配置和 UI 条件渲染，不涉及状态管理架构变更。

- 数据配置: ~80 行（5 个时代的默认值）
- UI 逻辑: ~60 行（处理函数 + 条件渲染）
- 总计: ~140 行
