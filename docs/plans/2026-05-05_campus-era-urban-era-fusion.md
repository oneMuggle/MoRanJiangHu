# 校园纪元 × 都市纪元融合计划

> 日期：2026-05-05
> 状态：待实施
> 前置：无
> 范围：校园与都市子纪元的融合预设、天赋组合、背景交叉

---

## 一、目标与背景

在现有 `contemporary_campus`（校园）和 `contemporary_urban`（都市）两个子纪元基础上，构建**跨场景融合预设**，允许玩家同时体验校园与都市生活，或在同一游戏中切换场景。

### 核心设计原则

1. **场景互通**：校园与都市场景可自由切换
2. **资源共享**：天赋、背景、气运可跨纪元组合
3. **叙事连贯**：都市通勤与校园生活在同一世界线下自然过渡
4. **NSFW 统一**：校园与都市的 NSFW 系统共享同一欲望框架

## 三、场景融合

### 3.1 新增跨场景开场

| 场景 ID | 名称 | 描述 |
|---------|------|------|
| `campus_urban_1` | 校园后街 | 连接校园与城市的商业街，奶茶店、网吧、小旅馆 |
| `campus_urban_2` | 地铁早高峰 | 从郊区宿舍到市中心公司的通勤 |
| `campus_urban_3` | 大学城夜市 | 学生与白领交汇的夜市 |
| `campus_urban_4` | 校外出租屋 | 合租生活，校外据点 |
| `campus_urban_5` | 校园旁自习室 | 共享自习室，可遇到各路人马 |

### 3.2 场景切换规则

- 主线剧情可在校园 ↔ 都市间自由切换
- 特定天赋/气运影响场景可及性
- NSFW 场景根据当前地点（校园/都市）使用对应欲望框架
